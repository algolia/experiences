# Algolia Experiences

Monorepo for Algolia Experiences infrastructure.

## Packages

| Package | Name | Description |
|---------|------|-------------|
| `packages/experiences` | `@algolia/experiences` | Evergreen loader script embedded via `<script>` tag |
| `packages/runtime` | `@experiences/runtime` | Runtime bundle containing InstantSearch and experience interpreter |
| `apps/resolver` | `@experiences/resolver` | Edge API that resolves experience IDs to bundle URLs |
