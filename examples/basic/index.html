Algolia Experiences - Basic Example

This example demonstrates how to load Algolia Experiences using the loader script.

Open the browser console to see the loader output.

Usage

<script src="https://github.com/algolia/experiences/releases/download/canary/experiences.js?appId=YOUR_APP_ID&apiKey=YOUR_API_KEY&experienceId=YOUR_EXPERIENCE_ID"></script>

Parameters

  • appId - Your Algolia Application ID
  • apiKey - Your Algolia API Key
  • experienceId - The ID of the experience to load
