# Basic Example

A minimal example showing how to use the Algolia Experiences loader.

## Setup

1. Update `index.html` with your Algolia credentials:
   - `appId` - Your Algolia Application ID
   - `apiKey` - Your Algolia API Key
   - `experienceId` - The ID of the experience to load

2. Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

3. Open http://localhost:5173 in your browser.
